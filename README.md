# Hello Python Project

This project is a simple Python application that prints "Hello, World!" when executed.

## Project Structure

```
hello-python
├── src
│   └── hello
│       ├── __init__.py
│       └── __main__.py
├── tests
│   └── test_main.py
├── pyproject.toml
├── requirements.txt
├── .gitignore
└── README.md
```

## How to Run

To run the program, navigate to the project directory and execute the following command:

```
python -m src.hello
```

This will print "Hello, World!" to the console.

## Testing

To run the tests, you can use the following command:

```
pytest tests/test_main.py
```

Make sure you have `pytest` installed in your environment.

## Dependencies

This project does not have any external dependencies. However, you can manage your dependencies in the `requirements.txt` file if needed.
